# Cloud Security Portfolio

## About This Portfolio

This portfolio documents hands-on cloud security projects across AWS, Microsoft Azure, and Google Cloud. Each project demonstrates identity security, logging, monitoring, threat detection, remediation, and incident response documentation.

## Portfolio Projects

### 1. AWS Cloud Security Detection & Response Lab

This project focuses on building a secure AWS lab environment using IAM, CloudTrail, GuardDuty, Security Hub, and S3 security controls.

Skills demonstrated:

- Identity and access management
- Cloud logging
- Threat detection
- Security findings review
- Misconfiguration remediation
- Incident response documentation

### 2. Azure Zero Trust Security Operations Lab

This project focuses on building a Microsoft Azure security lab using Microsoft Entra ID, Defender for Cloud, Microsoft Sentinel, and Log Analytics.

Skills demonstrated:

- Zero Trust concepts
- Azure identity security
- Cloud security posture management
- SIEM investigation basics
- Security recommendations review
- Incident documentation

### 3. Google Cloud Security Posture & Data Protection Lab

This project focuses on securing a Google Cloud environment using IAM, Cloud Audit Logs, Security Command Center, and Cloud Storage security controls.

Skills demonstrated:

- Google Cloud IAM
- Audit logging
- Storage security
- Security posture review
- Least privilege access
- Cloud remediation planning
